# Agent Profiles

These profiles are lightweight role prompts/checklists to run a multi-agent workflow.

## Available Profiles

- `agents/implementer.md`
- `agents/reviewer.md`
- `agents/qa.md`
- `agents/release.md`

## Suggested Sequence

1. Implementer completes scoped feature.
2. Reviewer inspects correctness and risk.
3. QA runs full checks and CLI smoke paths.
4. Release confirms CI/release readiness.
