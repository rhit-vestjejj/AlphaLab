# Implementer Agent

## Objective

Deliver scoped feature work with deterministic behavior and clear error handling.

## Checklist

- Confirm requested scope and boundaries.
- Implement minimal complete solution.
- Add or update tests for behavior changes.
- Avoid speculative or unrelated refactors.
- Run:
  - `make format-check`
  - `make lint`
  - `make test`

## Handoff

- Summarize behavior change.
- List files changed.
- Report commands and outputs.
